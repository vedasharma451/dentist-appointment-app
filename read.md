# Dentist Appointment Booking App

This is a simple full-stack project I built to practice React and Node.js.  
It allows users to view available dentists and book appointments online.

## Live Links
Frontend (Netlify):  
https://glowing-caramel-86d0e8.netlify.app
Backend (Render):  
https://dentist-appointment-app.onrender.com

## What I implemented
Display list of dentists from backend API  
Book appointment with basic form  
Send data using Axios (POST request)  
Show all appointments in admin panel  
Deployed frontend and backend separately  

## Tech Used
Frontend:
React
Axios
Bootstrap
Backend:
Node.js
Express
Deployment:
Netlify (frontend)
Render (backend)

## How to run locally
Backend:
cd backend  
npm install  
node server.js  
Frontend:
cd frontend  
npm install  
npm start  

## Notes
Appointment data is stored temporarily (no database used)  
Backend may take a few seconds to respond initially (Render free tier)
